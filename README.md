## The Stack

- **Core:** Pure HTML5, CSS3 (Modern Grid/Flexbox)
- **Design:** Inter & System Sans-Serif, Grainy Gradients, Glassmorphism
- **Motion:** Intersection Observer API for reveal animations
- **Philosophy:** "Anti-social, Anti-productive, Pro-focus."

## Design Principles

1. **Lowercase by Default:** We value the lack of pretense in lowercase typography.
2. **Asymmetry:** Grid layouts that breathe and force the eye to move.
3. **Monochrome + Contrast:** High-contrast interfaces (Pure Black `#0a0a0a`) with dimmed secondary information.
4. **Motion with Purpose:** Transitions are used to guide the user, not to distract them.
